# Глава 2

Для примеров из этой директории у вас должен быть установлен проект [BCC].

* `hello.py` — простой пример, выводящий trace-сообщения через kprobe.
* `hello-file.py` — похожий простой пример, подключённый к tracepoint системного вызова.
* `hello-map.py` — знакомство с концепцией BPF map.
* `hello-buffer.py` — использование ring buffer для передачи данных в user space.
* `hello-file-ring-buffer.py` — аналог `hello-file`, но с передачей данных через ring buffer.
* `hello-tail.py` — простой пример использования tail calls в eBPF.
